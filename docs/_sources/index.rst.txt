py-SolarCalc
============

About
-----

This project is a modernisation, optimisation and documentation project for `[SolarCalc] <https://www.ars.usda.gov/research/software/download/?softwareid=62>`_.
This aims to provide more understandable documentation, terse source, and more optimised modules.

It is currently a work in progress.

Requirements
------------
 * numpy
 * scipy
