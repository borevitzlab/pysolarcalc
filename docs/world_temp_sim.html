static read_deltat()[source]§

reads temperature deltas from the “deltat.gzip” data file. temperatures are in degrees centigrade * 10.0 apparently. the data file has headers that should be length 8 and contain the following info headers: grid_sz, xmin, ymin, xmax, ymax, n_cols, n_rows, n_months

the most important of these are n_months, n_rows, and n_cols

Returns:array of shape (nrows, ncols, n_months) of temperature deltas
Return type:np.ndarray
